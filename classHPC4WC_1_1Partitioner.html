Detailed Description

Space partitioning using a cartesian grid.

Uses MPI to exchange halo data.

Create a new Partitioner.

◆ scatter()

void HPC4WC::Partitioner::scatter ( )

Scatter the global field across all ranks.

Using the cartesian grid, distributes the global field from rank 0 to all ranks (where each rank gets the corresponding part of the global field).

◆ gather()

void HPC4WC::Partitioner::gather ( )

◆ applyPeriodicBoundaryConditions()

void HPC4WC::Partitioner::applyPeriodicBoundaryConditions ( )

Halo Exchange.

Applies periodic boundary conditions (halo exchange) in both i and j direction. See PeriodicBoundaryConditions::apply with PeriodicBoundaryConditions::PERIODICITY::BOTH. If only one rank is available, this function will fallback to PeriodicBoundaryConditions.
