Detailed Description

A field. Holds data.

Indices start at 0 and go up to nx + 2 * num_halo, this means, the actual field starts at (num_halo, num_halo)

void HPC4WC::Field::setFrom ( const Field &  f,
Field::const_idx_t &  offset_i,
Field::const_idx_t &  offset_j 
)

Set part of the field from another, given an offset.

This routine skips halopoints, meaning offset 0/0 places the data in the actual data field, not the halo. Internally relies on Field::setFrom(const Eigen::MatrixXd,const_idx_t&, const_idx_t&, const_idx_t&) and therefore might throw more errors if the offset's are off.

Exceptions
std::logic_errorIf the k dimension does not match.
Parameters
[in]fThe field to get the data from.
[in]offset_iThe offset in i direction.
[in]offset_jThe offset in j direction.
